## Experiment 3.2: Be Creative!

For the creative part of the server, I implemented a Unique Nickname Enforcement logic. If a new user registers with a nickname that is already active, the server will automatically close the old connection and replace it with the new one. This prevents ghost users and ensures that each nickname is associated with exactly one active session.
